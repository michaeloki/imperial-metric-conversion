### Reference Documentation
A docker-compatible springboot project.

Metric-imperial conversions for temperature, volume, area, weight and distance.
Temperature conversion pairs:
Celsius - Fahrenheit

Distance conversion pairs:
Kilometer - Mile
Yard - Meter
Inch - Centimeter
Foot - Meter

Weight conversion pairs:
Kilogram - Pound
Gram - Ounce

Fluid volume conversion pairs:
Pint - Liter
Liter - Gallon

Area conversion pairs:
Acre - Hectare
Meter - Acre


The insomnia project has the API endpoints for making the calls.
Junit was used to write the tests.
